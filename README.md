# Sistema de Inventario E-commerce

Para este proyecto armé el servidor usando Node.js con Express y trabajé con MySQL para la base de datos. Para conectar la base de datos con el código usé la librería Sequelize, que me ayudó a crear la tabla de productos automáticamente desde el mismo código usando la sincronización. En el modelo de la tabla configuré que el ID se genere solo de forma autoincremental para que no haya números repetidos, y le puse validaciones para que no acepte campos vacíos y que el stock nunca pueda ser menor a cero. 

Para la lógica de las compras, hice que el controlador primero verifique si todavía hay stock disponible. Si hay, usa la función decrement de Sequelize para restar exactamente una unidad directo en la base de datos; si el stock está en cero, la compra se bloquea y manda un error para no tener números negativos. 

Toda la parte visual la hice en un solo archivo HTML usando JavaScript y CSS para darle un diseño oscuro con detalles en colores neón. Ahí armé la tabla que muestra los datos que vienen de MySQL, un modal oculto para registrar o editar los productos, y los botones para eliminar y vender. También le agregué unas tarjetas en la parte de arriba que van sumando todo el dinero de las ventas y cuentan cuántos productos se quedaron sin stock o tienen cinco unidades o menos. Finalmente, usé la librería SweetAlert2 para que las alertas de confirmación, los errores cuando no hay stock y los avisos al guardar salgan como ventanas modernas y no como los mensajes normales que trae el navegador. Eso es exactamente todo lo que se armó y usó en el código.
